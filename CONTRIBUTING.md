# Contributing to Ossus

Thanks for helping make this dataset more accurate! Here's how.

## Fix a planet

1. Find the planet in `data/planets.json`
2. Make your correction
3. Open a **Pull Request** with:
   - What you changed
   - Your source (Wookieepedia link, film/series reference, etc.)

## Add a missing planet

Open an **Issue** with:
- Planet name
- Region
- Source (Wookieepedia URL)

## Report an error

Open an **Issue** with the label `data-error` and describe the problem.

## Rules

- Canon sources take priority over Legends
- Always link your source
- One correction per PR (easier to review)
- Respect the schema defined in `SCHEMA.md`
